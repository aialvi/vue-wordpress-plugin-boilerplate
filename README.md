# Aminul's Page Ranks

## Description

This is a WordPress plugin built with Vue 3 and WordPress.

## Installation

1. Upload the plugin files to the `/wp-content/plugins/aialvi-page-ranks`
   directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings in the admin panel.

## Development

To start developing the plugin:

1. Clone the repository.
2. Run `npm install` to install JavaScript dependencies.
3. Use `npm run dev` to start the development server with Vite.
4. For production builds, use `npm run build`.
